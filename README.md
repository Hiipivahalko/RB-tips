# Asennusohje

Kloonaa projekti
```
git clone https://github.com/Hiipivahalko/RB-tips.git
```

Suorita komento
```
gradle build
```

Ohjelma käynnistyy komennolla
```
java -jar build/libs/RB-tips.jar
```
